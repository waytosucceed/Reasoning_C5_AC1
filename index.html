CODING-DECODING

Coding:

Coding is a technique of representing words, phrases, or sentences in a secret format to prevent direct understanding.

Decoding:

Decoding is the process of deciphering the coded language and converting it back into the original format to understand the hidden message.

Important Tips to Solve Coding-Decoding

  1. Memorise the values of A-1, B-2, C-3, D-4,..., Y-25, Z-26.
  2. To find the reverse value of any alphabet simply, subtract the place value of that alphabet from 27.
    Example: The reverse of A (=1) is 27-1 = 26 = Z.

Types of Coding-Decoding


Substitution Coding: Specific words are replaced with predefined names or terms.
Example:
If COMB means PEN, PEN means SPOON, SPOON means CAR, and so on, what is used to eat food?
Solution: We use SPOON to eat food. But SPOON is coded as CAR, so the answer to this question is CAR.



Letter Coding: Letters are used in various directions (forward, backward, or both) to form a code
Example:
RED is coded as QDC, then what should be the code for BLUE?


Solution:RED is coded as QDC.Observe that it is coded by decreasing 1 for each alphabet.
So, in the same way, to code BLUE,
each alphabet should be decreased by 1,
i.e., B=A, L=K, U=T, E=D




Number Coding: Words are assigned numerals.
Example:
If CUP is coded as 389 and MEN is coded as 672, then what would be the code for PEN?
Solution: The code of each alphabet is already specified in the question itself, i.e., C=3, U=8, P=9, M=6, E=7 and N=2.
Using these codes, “PEN” is coded as 972.
